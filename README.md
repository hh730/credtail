# CredTail
### Setup
1. Open Terminal and navigate to project
2. Run npm i
3. cd ios
4. pod install
5. npx react-native run-ios (for iOS)
6. npx react-native run-android (for android)
#### This project is a React-Native app
The app is a Credtail PArtner App
